> **P52:**
>
> RTT 0时，窗口长度W/2
> RTT 1时，窗口长度W/2 + α* W/2 = (1 + α) *W/2
> RTT 2时，窗口长度(1 + α)W/2 + α(1 + α)W/2 = (1 + α)^2 *W/2
> RTT 3时，窗口长度(1 + α)2W/2 + α(1 + α)2W/2 = (1 + α)^3 *W/2
> ...... RTT n时，窗口长度((1 + α)^n *W/2
> 设n时达到W，此时((1 + α)^n *W/2 = W
> n = log2(1 + α)
> 因此，当log2(1 + α) * RTT时到达W，与吞吐量无关
